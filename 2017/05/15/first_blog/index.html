用hexo搭建网站以及配置发布

发表于 2017-05-15

如何开始

之前看到过很多github pages搞出来的个人网站，还不需要申请域名，就也想自己来搞搞，又了解到除了wordpress之外还有个node写的博客发布程序，就心痒痒，想试试，就开始搞起了hexo

第一步准备

先去github上面建个仓库，new repository

名字必须的 你的用户名.github.io 这样的方式 比如 jesseband.github.io
其他都默认，然后创建即可

安装hexo

sudo npm install -g hexo-cli
hexo-cli 是hexo的命令行模块
新建个项目目录，执行hexo init命令
hexo new name
然后在source目录下边_posts里面会有个name.md文件，在这里吗就可以直接写你的博客了，当然是可以用markdown语法来写了
hexo generate 执行这个命令生成静态文件
本地是可以预览效果的 hexo server -p 4000 默认是4000端口，也可以自定义修改

配置发布

在根目录下面，就是执行init的目录下边会有个_config.yml文件，这个就是配置文件
在最下边找到deploy 的配置行
这样配置


deploy:
     type: git
     repo: https://github.com/jesseband/jesseband.github.io.git
     branch: master

然后安装 hexo-deployer-git
sudo npm install hexo-deployer-git –save
安装完就可以执行hexo deploy将public里面的文件发布到github项目中了，然后用jesseband.github.io就可以访问看到效果了。

hexo主题

github上都有不同风格的主题包可以下载，解压缩放到theme文件下边即可，在_config.yml文件中配置上相对应的名字即可

以上就是我简单用hexo搭网站的简单总结和记录，更复杂的hexo功能还有待去开发。
